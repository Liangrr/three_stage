安装
npm install <包名> 
简写：npm i <包名> 

全局安装
npm install <包名> -g

安装的包是代码依赖/生产依赖
npm install <包名> --save/-S

安装的包是开发依赖
npm install <包名> --save-dev/-D




卸载
npm uninstall <包名>
